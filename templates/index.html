Self-hostable media server

How it works

You can upload files to your instance by going to the web gui located at /upload, you can also upload files from a script by sending a post request to /api/upload. We are constantly working on ways to improve this.

Software Credits

This server is powered by A3D CDN, a self-hostable media server written in python deployed within 1 minute using docker and docker compose. Check out our repository on github and consider deploying it yourself.
